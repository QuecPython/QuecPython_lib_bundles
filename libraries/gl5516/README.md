# gl5516

​	注意：光敏电阻值的计算跟接入电路有关！

**类引用：**

```python
from gl5528 import Gl5528
```

 

**实例化参数：**

| 名称    | 必填 | 类型    | 说明       |
| ------- | ---- | ------- | ---------- |
| adc_dev | 是   | ADC对象 | adc设备    |
| adcn    | 是   | ADC通道 | 例ADC.ADC0 |

```
AdcDevice = ADC()
gl5516=Gl5516(AdcDevice,ADC.ADC0)
```

**接口函数：**

l **read()**

​	读取光敏电阻值。

​	备注：不同接线光敏电阻值计算方式可能不一致，此方法只适合我们的600s系列。

参数：

​    无。

返回值：

| 名称       | 类型 | 说明   |
| ---------- | ---- | ------ |
| resistance | 数值 | 电阻值 |

l **read_volt ()**

​	读取电压值。

​	用于计算光敏电阻值。

参数：

​    无。

返回值：

| 名称 | 类型 | 说明 |
| ---- | ---- | ---- |
| volt | 数值 | 电压 |
