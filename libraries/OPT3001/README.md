**接口函数：**

l **set_measure_mode(mode)**

​	设置测量模式，持续测量和单次测量或shutdown。

参数：

| 名称 | 必填 | 类型 | 说明                                              |
| ---- | ---- | ---- | ------------------------------------------------- |
| mode | 否   | int  | 默认2：连续测量；<br />还可选0：关机，1：单次测量 |

返回值：

​       -1 ： 失败

​		0：  成功

l **read ()**

​	读取照度值。

​	注意转化时间初始化为800ms，两次读取之间间隔需大于800ms。

参数：

​    无。

返回值：

| 名称 | 类型 | 说明     |
| ---- | ---- | -------- |
| lux  | int  | 光照度值 |
